  • Contoh link / web phising gimana?

To the point aja, jadi contohnya kaya text disamping ini m.youtube.com (LINK DISEBELAH KIRI GA ADA APA APA / GA MEMBAHAYAKAN SAMA SEKALI.) contohnya kek disamping, kalo di pencet bisa ga sesuai sama tampilan link.

cara kerja phising:
Saat tampilan link dianggap betul / benar / ga ada yang dicurigain, tapi pas dipencet beda sama thumbnail. Ini bisa aja kalo di bahasa YouTube itu "clickbait". Jadi saat dipencet link clickbait, virus / (tambahan kode) bisa mulai nyuri data privasi kalian, kaya foto, video, hack kamera dll. Apalagi kalo link clickbait diisi virus, jadi kita sama sekali ga bisa ngapa ngapain. Dan Cara ngatasinnya yaitu langsung matiin hape terus wipe data (cara ini dilakukan jika sudah sangat brutal)
